# پنداریار پایدار — قالب Next.js

بازسازی محتوایی صفحه فرود پنداریار پایدار (مشاوره کسب‌وکار، فروش و بازاریابی، منابع انسانی) با Next.js 14 و Tailwind CSS، راست‌به‌چپ با فونت Vazirmatn.

## اجرا

```bash
npm install
npm run dev
```

سپس `http://localhost:3000` را باز کنید.

## بخش‌ها

هدر، هیرو، حوزه‌های تخصصی (۳ کارت خدمات)، از استراتژی تا اجرا، آمار (مشتریان/کمپین‌ها/تعاملات/قراردادها)، پرسش‌های متداول، فرم تماس + اطلاعات دفتر، همراهان ما (لوگوی مشتریان)، درباره ما، فوتر.

## نکات

- لوگوی واقعی، عکس‌ها و لوگوی مشتریان با پلیس‌هولدر جایگزین شده‌اند؛ آن‌ها را در `public/` قرار داده و در `page.js` جایگزین کنید.
- نقشه در بخش تماس فعلاً پلیس‌هولدر است؛ می‌توانید iframe نقشه گوگل را جایگزین کنید.
